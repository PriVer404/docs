# Getting Started

<Info>
  ContractKen Word Add-in will work with Microsoft 365 versions later than 16.0.11629 on Windows 10, and Word Online
</Info>

<Steps>
  <Step title="Install Addin Inside Word" stepNumber={0} titleSize="h3">
    There are 2 options to install the add-in.

    1. Launch ContractKen Word add-in\
       with an existing .docx file

       [Open In Word](https://www.contractken.com/word#)
    2. Alternatively, install it from [Microsoft Appsource](https://appsource.microsoft.com/en-us/product/office/WA200003523)
  </Step>
  <Step title="Open A Document In Word" icon="book-open" stepNumber={2} titleSize="h3">
    Open the word document that you want to work on, or a blank document if you want to draft a contract.
  </Step>
  <Step title="Launch The ContractKen Add-In" titleSize="h3">
    The addin will be added to your Add-Ins and will be available in the Home Section of the top panel.

    <Note>
      If you opened the add-in directly through the link above, you can ignore this step.
    </Note>
    ![Addiniconlocation Pn](/images/addiniconlocation.png)
  </Step>
  <Step title="Login / SignUp" titleSize="h3">
    <img
      src="/images/login-signup.png"
      alt="Login Signup Pn"
      title="Login Signup Pn"
      className="mr-auto"
      style={{ width:"26%" }}
    />

    Login if you have created an account already or SignUp to create a new account.
  </Step>
  <Step title="Choose Service" titleSize="h3">
    <img
      src="/images/image.png"
      alt="image.png"
      title="image.png"
      style={{ width:"29%" }}
    />

    There are 3 options:

    <CardGroup cols={2}>
      <Card title="Review Contract" icon="sparkles">
        Run various tools and analysis on a fresh contract (no redlines)
      </Card>
      <Card title="Draft a New Contract" icon="sparkles">
        - Upload a precedent and change instructions to redraft the precedent into a new contract
      </Card>
      <Card title="Review Redlines" icon="sparkles" href="https://contractken-66333041.mintlify.app/review-redlines">
        - If you have a redlined contract, you can get an extensive Redline Report detailing each change
      </Card>
    </CardGroup>
  </Step>
</Steps>
